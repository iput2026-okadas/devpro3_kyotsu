# CSV ビューア E2E テストシナリオ

Playwright の Chromium から Flask 製CSVビューアへアクセスし、次を確認します。

1. 一覧のうち最新のCSVが初期選択され、列、行、件数が表示される。
2. 一覧から別のCSVを選択すると、URL、選択状態、表示内容が切り替わる。
3. 気温列をクリックすると、数値として昇順・降順に並べ替えられる。
4. CSV・JSONエクスポートで、選択中のファイル名と表示データが出力される。
5. 存在しないCSVを指定すると、エラーメッセージが表示される。

テストでは `tests/fixtures/csv_viewer/` のCSVを一時ディレクトリへコピーして使用し、
`Server/data/` のサンプルデータは変更しません。FlaskサーバーとChromiumはテスト中に
自動で起動し、終了時に停止します。

画面を見ながら確認する場合は、`E2E_HEADLESS=false` と `E2E_SLOW_MO` を指定して
pytestを実行します。詳細なコマンドは `tests/README.md` を参照してください。
